# University Admission Chatbot

This project implements a faculty-focused chatbot use case:

## 11. University Admission Enquiries

Problem: Students repeatedly ask about admission procedures, eligibility, and deadlines.

Chatbot Solution: A chatbot provides instant answers about courses, fees, admission requirements, required documents, and admission timelines.

## What The Bot Handles

- Admission procedure steps
- Course details and fee structure
- Eligibility criteria
- Admission start and deadline dates
- Required documents
- Official admission contact details
